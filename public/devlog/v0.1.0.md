# v0.1.0 — 项目初始化

> 2026-03-11

## 已实现功能

- **落地页**：展示 "Piecky is coming..." 品牌标语
- **关于页面**：项目介绍、GitHub 地址、Telegram 联系方式
- **中英文切换**：支持 zh / en 两种语言，选择会记住到本地存储
- **深色模式**：自动跟随系统配色方案
- **Cloudflare Workers 部署**：通过 OpenNext 打包，一键部署到 Cloudflare
- **Tauri 桌面应用**：支持打包为 macOS ARM（M 系列芯片）原生应用
- **更新日志页面**：展示项目迭代记录
- **想法页面**：以卡片形式展示个人想法与灵感

## 技术栈

- Next.js 16 + React 19 + Tailwind CSS 4
- Tauri 2（桌面端）
- OpenNext + Cloudflare Workers（Web 端）
- pnpm 包管理
